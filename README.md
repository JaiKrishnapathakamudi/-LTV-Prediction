\# Customer Lifetime Value (LTV) Prediction



\## Overview



A machine learning-powered web application that predicts Customer Lifetime Value (LTV) and helps businesses identify high-value customers, improve retention strategies, and optimize marketing campaigns.



\## Features



\* Customer Lifetime Value Prediction

\* RFM (Recency, Frequency, Monetary) Analysis

\* Customer Segmentation

\* Data Cleaning \& Preprocessing

\* Interactive Data Visualization

\* PDF Report Generation

\* User Authentication System

\* Google Login Integration



\## Technologies Used



\* Python

\* Flask

\* Pandas

\* NumPy

\* Scikit-learn

\* Matplotlib

\* SQLite

\* HTML/CSS/JavaScript



\## Project Structure



\* app.py – Main Flask application

\* ltv\_prediction.py – LTV prediction logic

\* advanced\_model.py – Advanced ML models

\* rfm\_analysis.py – Customer segmentation

\* data\_cleaning.py – Data preprocessing

\* visualization.py – Charts and graphs

\* report\_generator.py – PDF report generation

\* auth.py – Authentication system

\* database.py – Database operations



\## Installation



```bash

pip install -r requirements.txt

python app.py

```
